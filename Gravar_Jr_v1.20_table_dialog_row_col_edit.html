v1.20

Video shows the current filename or source title. Use the folder button to choose a video source and access project files. Choosing Local file, YouTube, or Panopto immediately starts that loading path.

Use Start and End to copy the current video position into the selected transcript line. Click the line-number area, or touch-hold it for 0.5 seconds, to move the video to that line's Start time. Double-click or double-tap the line-number/check area to show editable Start time fields for all transcript lines; double-click or double-tap any visible timecode to hide them. Add creates a line below the selected line. Field Note opens a project-wide rich-text field note with compact Gravar-style formatting buttons, table insertion with chosen row/column counts, table row/column editing, and RTF export; the ▣ icon opens a line-specific note under the transcript row.

The diagnostic bar beneath the video also functions as a timeline. Drag on the video to scrub ±2 seconds. Hold Play for one second to show playback speed choices.
