Getting Started with Gravar Jr.

v1.23

1. Choose your class and video

Open the File menu. For class-assigned streaming media, enter the class code (for example cds670-f26) and choose Set. Gravar Jr. remembers the class on this device. A launch link such as ?class=cds670-f26 selects the class automatically. Open Streaming Files and choose an assigned title. Use Refresh catalog to request the newest GitHub catalog. If GitHub is temporarily unavailable, Gravar Jr. uses the last catalog that loaded successfully on this device.

For a video that is not in the class catalog, choose Local, YouTube, or Panopto directly from the File menu. Local files remain on your device. YouTube and Panopto require a web-served copy of Gravar Jr.; some videos can still prohibit embedding.

2. Control playback

Use the center Play/Pause button. The ±1s and ±5s buttons jump by the labeled amount. Press and hold a jump button to step repeatedly in small frame-sized increments. Press and hold Play for one second to open playback-speed choices. Drag horizontally across the video to scrub within approximately ±2 seconds of the point where the drag began. Drag or tap the timeline strip under the video to move through the full video.

3. Create transcript lines

A blank transcript line is available automatically. Enter a short Speaker ID in the Speaker field and type the transcript in the large text field. Choose Add to insert a new line below the selected line. Choose Delete to remove the selected line. Click or tap any transcript field to select its line. Toggle Follow on to highlight the transcript line whose Start time is the latest Start at or before the current video position. The followed line is shown in pale yellow and changes automatically as the video moves.

4. Set Start and End times

Move the video to the beginning of an event and choose Start. Move to the ending point and choose End. A red check indicates that a time has been stored. Single-click or tap the line-number/Start-check area to seek to that line’s Start time; touch-hold the same area for about half a second also seeks to Start.

5. Show and edit all timecodes

Double-click or double-tap the line-number/red Start-check area on any line. Gravar Jr. reveals editable Start timecodes to the left of every Speaker field and editable End timecodes to the right of every transcript field, before the Line Note/coding rail. Edit any value using seconds, mm:ss.t, or hh:mm:ss.t. Press Enter or leave the field to commit the edit. Clear a field to remove that time. Double-click or double-tap the line-number/Start-check area again to hide all timecodes.

6. Add Line Notes

Choose the ▣ icon in a line’s right-hand rail to open its Line Note beneath that transcript row. Type freely; spaces remain in the note and never control video playback. The note is stored with that transcript line and saved in the project.

7. Use the rich-text Field Note

Choose ◇ Field Note below the transcript. Use B and I for bold and italics, the color dots for text color, ▦ to insert a table, +R/−R to add or delete rows, +C/−C to add or delete columns, and × to clear selected formatting. Click inside a table cell before using row or column editing commands. Spaces typed in the Field Note remain in the note and never toggle playback.

8. Create, save, and reopen working projects

Choose New Project, give the project a name, and name its project folder. Every project receives an internal Gravar Jr. project folder in the browser’s local project library. Where the browser provides writable folder access, choose Create Device Folder…, select a parent location, and Gravar Jr. creates or links the named folder there. Choose Save Project to update the same working project rather than downloading another timestamped copy. Choose Load Project to reopen a project from the local library.

If a writable device folder is linked, Save Project also writes the current project JSON into that folder, and exports are written there. If folder access is unavailable, the internal project library still works and exports use the browser’s normal file-save/download behavior. Local video files may need to be reselected after reopening because browser security can restrict persistent access to local media.

9. Import and export portable files

Choose Import Project to bring a portable Gravar Jr. JSON file into the local project library. Open Export › for Export Project JSON, Export CSV, Export HTML, or Export Field Note RTF. Project JSON is the portable backup/transfer format; CSV is useful for analysis workflows; HTML creates a readable report; and RTF preserves the Field Note’s formatted text and table content.

File menu and video sources

Custom File icon — click/tap
Opens or closes the File menu without changing the current project.
Local
Opens the device file picker for a browser-playable video. The project remembers the filename but cannot retain permission to the local file after the page closes.
YouTube
Prompts for a YouTube URL or video ID and loads the official embedded player when the page has a valid web origin.
Panopto
Prompts for a Panopto embed URL. Browser cross-origin security may prevent Gravar Jr. from reading Panopto playback time automatically.
Class / Set
Saves the normalized class code locally and loads only that class’s catalog. A ?class=... launch parameter overrides the remembered class.
Streaming Files ›
Shows only the current class catalog. Choosing a title loads its URL according to its catalog source type: YouTube, Panopto, or direct browser-playable URL.
Refresh catalog
Fetches the current class JSON again with browser caching disabled. The last successfully fetched catalog is stored locally as fallback.
New Project
Opens a project setup dialog. Name the project and its internal project folder; where supported, you may also create/link a writable device folder before creating the project.
Load Project
Opens the local Gravar Jr. project library on this device. Open a saved working project or delete an unwanted library entry.
Save Project
Updates the current working project in the local project library. If a writable device folder is linked, it also writes the current project JSON there.
Project Folder ›
Shows the current folder status. Create/link a writable device folder where supported, or disconnect the current folder. The internal Gravar Jr. project folder always remains available.
Import Project
Reads a portable Gravar Jr. JSON file and adds it to the local project library as a working project.
Export ›
Opens portable Project JSON, CSV, HTML report, and Field Note RTF export commands. Exports go to the linked writable folder when available; otherwise the browser handles the file save/download.

Video and transport controls

Play/Pause — click/tap
Toggles playback.
Play — long press (1 second)
Opens playback speed choices from 0.5× to 2×.
↶5s, ↶1s, 1s↷, 5s↷ — click/tap
Jump backward or forward by the labeled number of seconds.
Jump buttons — long press
Repeatedly steps in approximately 1/30-second increments until released.
Video image — horizontal drag/scrub
Scrubs up to approximately two seconds backward or forward from the drag starting point. This is the app’s horizontal swipe-like video gesture.
Timeline strip — tap/click/drag
Moves to a proportional position across the full video duration.
Vertical swipe
Scrolls the transcript list, Help/Getting Started content, or Field Note where that region is scrollable. There is no separate hidden swipe command.

Transcript controls and actions

Line number / red Start check — single click/tap
Selects the line and seeks the video to its Start time when one exists.
Line number / red Start check — long press
After about 0.5 seconds, seeks to the line’s Start time.
Line number / red Start check — double-click/double-tap
Toggles editable Start and End timecodes for all transcript lines.
Red timing check — hover or touch-hold
Shows the stored timing value. Start checks are in the line-number area; End checks appear at the right edge of the transcript while timecodes are hidden.
Start time field
Appears left of Speaker when timecodes are shown. Accepts seconds, mm:ss.t, or hh:mm:ss.t. Enter or leaving the field commits it; blank clears it.
End time field
Appears right of transcript text and before the Line Note/coding rail when timecodes are shown. Editing rules match Start.
Speaker
Editable three-character Speaker ID, automatically uppercased.
Transcript text
Editable multiline transcript. The field grows to fit its content.
Start / End buttons
Copy the current video position into the selected line’s Start or End time.
◇ Field Note
Opens the project-wide rich-text Field Note.
Add
Inserts a new transcript line immediately below the selected line.
Delete
Deletes the selected line; when only one line remains, it is cleared rather than removing the transcript completely.
Follow — click/tap
Toggles playback-following. When on, Gravar Jr. highlights in pale yellow the transcript line with the greatest Start time that is less than or equal to the current video time. Lines without Start times are ignored; before the first timed line, no line is highlighted.
▣ Line Note
Opens or focuses the line-specific plain-text note beneath that transcript row.

Field Note formatting and tables

B / I
Toggle bold or italic formatting for selected text or subsequently typed text at the caret.
Color dots
Apply red, blue, green, grey, or black to selected text.
▦ Table
Prompts for row and column counts and inserts a table at the current Field Note caret.
+R / −R
Add a row after the current table row or delete the current row. Deleting the last row removes the table.
+C / −C
Add a column after the current table column or delete the current column. Deleting the last column removes the table.
× Clear formatting
Removes formatting from selected Field Note text without deleting the text.
Field Note typing
All ordinary keys, including Space, remain in the editor. Table cells can be navigated with normal browser text-editing and pointer/touch actions.

Keyboard reference

Space
Play/Pause only when focus is not in an input, textarea, select, Line Note, Project Note, Field Note, contenteditable region, or other editable control.
Left / Right Arrow
Seek backward or forward 1 second when not editing text.
Shift + Left / Right Arrow
Seek backward or forward 5 seconds when not editing text.
N
In the preserved legacy annotation workflow, captures the current time as a new annotation Start and focuses its note field.
Enter in a visible Start/End timecode
Commits the edited time and keeps the field available for further editing.

Project and catalog behavior

Class catalogs are independent JSON files. By default the app resolves catalogs/<class-code>.json relative to the application, which is suitable for a GitHub Pages repository. The catalog base can also be changed by editing the gravar-jr-catalog-base meta tag to a raw GitHub directory URL. Only the selected class catalog is requested. Successful catalogs are cached under that class code.
